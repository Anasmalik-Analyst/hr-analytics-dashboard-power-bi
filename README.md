# HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard developed using Microsoft Power BI. It helps analyze employee attrition, workforce demographics, salary trends, and key HR metrics to support data-driven decision making.

---

## 📊 Key KPIs

* Total Employees
* Active Employees
* Attrition Count
* Attrition Rate
* Average Age
* Average Monthly Income

---

## 📈 Dashboard Visuals

* Attrition by Department
* Attrition by Gender
* Attrition by Job Role
* Attrition by Age Group
* Attrition by Education Field
* Average Monthly Income by Job Role
* Attrition by Overtime

---

## 🎛 Interactive Filters

* Department
* Gender
* Job Role
